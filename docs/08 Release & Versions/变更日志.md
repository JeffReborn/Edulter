## 2026-03-21

### 客户显示名唯一与「新建 / 更新画像」分流

- **数据**：`clients.displayName`（文档字段名 `display_name`）增加**全库唯一**约束；迁移 `0004_client_displayname_unique` 对历史重复数据去重后再建索引。
- **画像提取 `POST /api/profiles/extract`**：支持 `clientId`；有 `clientId` 为**更新**（仅按 id 定位），无则为**新建**。新建时 `clientDisplayName` 可选：填写须 **≥ 3** 字且唯一；不填则服务端**占位名**。显示名冲突返回 **409**（`CLIENT_DISPLAY_NAME_TAKEN` 等）。
- **改名 `PATCH /api/clients/:id`**：仅更新 `displayName`；校验规则与上一致；冲突 **409**（`DISPLAY_NAME_TAKEN` 等）。
- **前端**：首页「新建客户画像」；详情「更新客户画像」链到 `/profiles` 并带 `clientId`；客户列表显示名**内联编辑**；文案区分新建/更新。
- **文档**：已同步 [[PRD]]、[[客户画像提取模块]]、[[客户记录模块]]、[[API合同]]、[[数据模型]]、[[页面导航与演示流程]]、[[产品决策]]（决策 12，决策 11 标记替代）。
